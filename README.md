|---|---|---|---|
| `PersonalWebsite/` | V1 | 5173 | Warm cream — `#FFFBF2`, Space Grotesk, blueprint grid |

## Stack

- React 19 + Vite 8
- Tailwind CSS v4
- Framer Motion v12
- Space Grotesk (v1) / Inter (v2)

## Local Dev

```bash
# V1 — Warm
cd PersonalWebsite && npm install && npm run dev

# V2 — Dark
cd PersonalWebsite-v2 && npm install && npm run dev
```
